    short : -32768 to 32767

2) Purpose of comma in print statement
   In Java, comma is used to separate arguments in methods like printf(), but not in print() or println().
   Example:
       System.out.printf("Age = %d", age); 

3) Why class name and file name have to be same?
   In Java, the file name must match the public class name so the compiler and Java Virtual Machine can find, compile, and execute the class without confusion.
